# 列挙型

## 概要

定義方法は下記

```java

[修飾子] enum 列挙型名 { val1, val2, val3}

```

- クラス定義の中、列挙型の定義だけを記述したソースファイルで定義が可能
- メソッドの中で定義することはできない 

## Enumクラス

- java.lang.Enumクラスを継承したfinalクラスになる
- 列挙した値はpublic static final指定されたクラス変数の名前となる
    - 列挙値前後にダブルクォートはつけない
- valus()とvalueOf()というstaticメソッドが自動追加される
    - values()
        - 全ての列挙値を配列で返却
    - valueOf(String name)
        - 引数で指定された名前を持つ値を返却する
- 列挙型の実体は定数やメソッドを持つクラス
  - インスタンス化はできない
  - コンパイルするとclassファイルが生成される
  
## 列挙型でのコンストラクタ／変数／メソッド定義

- コンストラクタ、変数、メソッド定義が可能
- メソッドとしてabstractも利用可能

```java

enum Card {
    SPADES(3), CLUBS(1),
    DIAMONDS(4), HEARTS();

    private int a;
    Card(){
        this.a = 100; // コンストラクタ
    }

    Card(int a) {
        this.a = a; // コンストラクタ
    }

    public int getA(){
        return a;
    }
}

public class Main {
    public static void main(String[] args) {
        Card card = Card.SPACES;
        System.out.println(card); //SPADES
        System.out.println(card.getA()); // 3
        System.out.println(card.ordinal()); //0
        System.out.println(Card.HEARTS.getA()); //100

        for (Card obj : Card.values()) {
            System.out.println(obj + "") // SPADES CLUBS DIAMOND HEARTS
        }
    }
}

```
